# 🏥 Healthcare Readmission Predictor

A machine learning app that predicts the risk of hospital readmission 
within 30 days for diabetic patients.

## Live App
[Click here to try it](your-streamlit-url)

## Built With
- Python, Pandas, Scikit-learn, Streamlit
- Dataset: Diabetes 130-US Hospitals (UCI / Kaggle)
- Model: Logistic Regression with class balancing (54% recall)

## Project Structure
- `week1_exploration.ipynb` — Exploratory data analysis
- `week2_cleaning.ipynb` — Data cleaning and preprocessing  
- `week3_modeling.ipynb` — Model training and evaluation
- `app.py` — Streamlit web application
